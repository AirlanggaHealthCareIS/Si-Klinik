USER STORY "Menambah Data Obat"

Feature : Apoteker dapat menambah data obat baru

1. Scenario : Input data obat jika data benar dan lengkap

-Given  Apoteker sudah memilih menu inventory obat dan berada pada halaman form input data obat
-When   Apoteker mengisi form input data obat
-And 	Apoteker menekan tombol "OK"
-Then 	Apoteker dapat melihat dan mengecek inputan data obat yang muncul pada Table Review
-And 	Apoteker menekan tombol "SAVE"
-Then	Apoteker berada pada halaman form input data obat
-And	Apoteker melihat pesan "Data Berhasil Disimpan"
